# open-webui-cheat-sheet


# Install

## ubuntu
```shell
cd ~/Projects/ai/GUI
git clone https://github.com/open-webui/open-webui.git

pyenv local 3.11

python3 -m venv venv
source venv/bin/activate

pip install open-webui

open-webui serve
```
